## Mojava 升级到 Catalina 注意

可能已经有所了解，Apple 这次是铁了心的要干掉32位软件，所以 Catalina 及之后版本都不可能再支持 32位软件，这导致原本许多应用特别是游戏都无法在新系统上运行，而唯一的解决办法是等软件更新支持或者在新系统上运行虚拟机跑 Mojava 环境，所以你需要权衡带来的影响。
